# Todo-List-API

## Setup
1. Clone the repository:
```bash  
   git clone "https://github.com/Polyston/Todo-List-API.git"
   cd Todo-List-API
```
2. Using .env.example as a template, create a new .env file and add new values for each variable
   
3. To run the application:
```bash
   uvicorn main:app --reload
```
